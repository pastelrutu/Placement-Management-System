# Placement Management System

## Description
A menu-driven Placement Management System developed using Python and MySQL. The project helps manage students, companies, placement drives, applications, and placement reports.

## Features
- Student Management
- Company Management
- Placement Drive Management
- Application Management
- Selection Status Reports

## Technologies Used
- Python
- MySQL
- PyMySQL
- Git & GitHub

## Modules

### Module 1: Student Management
- Add Student
- View Student
- Update Student
- Delete Student

### Module 2: Company Management
- Add Company
- View Company
- Update Company
- Delete Company

### Module 3: Placement Drive
- Add Drive
- View Drive
- Update Drive
- Delete Drive

### Module 4: Application Management
- Apply for Company
- View Application
- Update Application
- Delete Application

### Module 5: Selection Status & Reports
- View Selected Students
- View Rejected Students
- View Shortlisted Students
- Total Applications
- Total Selected Students
